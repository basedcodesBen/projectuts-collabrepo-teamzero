List dan Table

HTML List

HTML list berguna untuk menggabungkan beberapa item yang berhubungan di dalam sebuah list.

Unordered HTML List

Unordered list dimulai dengan tag ul dan setiap item dalam list dimulai dengan tag li. Secara default item yand dilist akan ditandai dengan lingkaran hitam kecil, namun kita bisa menggantinya menjadi kotak hitam kecil atau tidak ada tanda sama sekali.

Berikut contoh penulisan dan hasil dari kode yang dijalankan.

Ordered HTML List

Ordered list dimulai dengan tag ol dan setiap item dalam list dimulai dengan tag li. Secara default item yang dilist akan ditandai dengan angka, namun kita bisa mengubahnya menjadi huruf romawi atau abjad.

HTML Description List

Description list adalah list yang berisikan syarat, dengan deskripsi pada setiap syaratnya. Tag dl mendefinisikan list deskripsi, tag dt mendefinisikan syarat (nama), dan tag dd mendefinisikan setiap syarat.
